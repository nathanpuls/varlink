# VARLINK

A minimalist dynamic link engine for power users.

## Features
- **Horizontal Header**: Streamlined navigation and search.
- **Variable Links**: Use `$` in URLs for dynamic replacements.
- **Quick Variables**: Set common values for instant access.
- **Reorderable**: Drag and drop to organize your vault.

## Run Locally

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deployment
This project is optimized for Cloudflare Pages.
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
